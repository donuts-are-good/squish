Welcome to SQUISH IRC SERVER
sweet.yachts

Supported commands:
 NICK <nickname>     - Set or change your nickname
 USER <username> * * :<realname>  - Set your username and real name
 JOIN #<channel>     - Join a channel (e.g., JOIN #general)
 PART #<channel>     - Leave a channel
 PRIVMSG <target> :<message>  - Send a message to a channel or user
 LIST                - List all channels
 NAMES #<channel>    - List users in a channel
 TOPIC #<channel> :<topic>  - Set or view channel topic
 MODE <target> <modes>  - Set user or channel modes
 QUIT :<message>    - Disconnect from the server

NickServ commands (use /msg NickServ <command>):
 REGISTER <password> <email>  - Register your nickname
 IDENTIFY <password>  - Identify with your registered nickname
 SET PASSWORD <new_password>  - Change your password
 INFO <nickname>      - Get information about a nickname
 GHOST <nickname> <password>  - Disconnect an old session

Default channels: #general, #help, #off-topic

Enjoy your stay and happy chatting!
